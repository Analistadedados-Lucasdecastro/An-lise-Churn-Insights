 Análise de Churn - Setor de Telecomunicações 📊

Este projeto utiliza **SQL (MySQL)** para analisar uma base de dados de uma empresa de Telecom, com o objetivo de entender os motivos que levam os clientes a cancelarem seus serviços (Churn).

## Descobertas Principais
- **Taxa de Churn:** 26% da base de clientes.
- **Contratos:** Clientes com contratos mensais têm uma propensão muito maior ao cancelamento.
- **Sensibilidade ao Preço:** Clientes que saem pagam, em média, R$ 74,44, enquanto os que ficam pagam R$ 61,27.

## Tecnologias Utilizadas
- **Banco de Dados:** MySQL
- **Linguagem:** SQL
- **Conceitos:** Limpeza de dados (ETL), Agregações, Subqueries e Lógica de Negócio.

## Conclusão
A recomendação estratégica é converter clientes de contratos mensais para anuais e revisar a faixa de preço acima de R$ 70,00, onde ocorre a maior taxa de desistência.
